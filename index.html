Macro Tales

Macro Tales is a web-based tabletop RPG assistant designed to help Dungeon Masters build, manage, and run large-scale campaigns for games like D&D, Pathfinder, and World of Darkness.

  • Interactive virtual tabletop for managing characters and combat
  • Supports both simplified and full character sheets for different play styles
  • Planned map maker and lore storage tools to centralize worldbuilding
